# 🧠 MindCare – AI Powered Mental Health Awareness Platform



An **AI-powered mental health awareness platform** designed to help users monitor emotional well-being using **Natural Language Processing and Machine Learning**.

MindCare analyzes user text input to detect **stress, anxiety, sadness, and positive emotional states** and provides awareness resources for better mental health.

---

# 📌 Project Overview

Mental health issues among students are increasing rapidly. Many people hesitate to seek professional help due to stigma or lack of awareness.

**MindCare** aims to bridge this gap by providing:

• AI-powered emotional analysis
• Mental health awareness resources
• Simple and accessible web platform
• Data-driven insights about emotional state

# ⚙️ Installation Guide

1️⃣ Clone Repository

2️⃣ Go to Project Folder

cd MindCare-AI-Powered-Mental-Health-Awareness-Platform
